| 支持的环境 | 不支持的环境 | 未知 |
| --- | --- | --- |
| .NET 4.7.2 | .NET 4.8 | .NET 3.5 |
| - | .NET 4.6.1 | .NET 4 Client Profile |
| - | .NET 4.6 | .NET 4 |
| - | .NET 4.5.2 | .NET 3.5 Client Profile |
| - | .NET 4.5.1 | .NET 3 |
| - | .NET 4.5 | - |
