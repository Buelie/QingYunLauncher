# 这是什么?
***这是一个基于.NET 4.7.2开发的Minercraft启动器,它的代名词是跨时代/极简!!!如果你想安装它请往下看***

# 安装
***安装清云启动器需要.NET环境,目前它支持的环境和不支持的环境我们放下表中了:***
| 支持的环境 | 不支持的环境 | 未知 |
| --- | --- | --- |
| .NET 4.7.2 | .NET 4.8 | .NET 3.5 |
| - | .NET 4.6.1 | .NET 4 Client Profile |
| - | .NET 4.6 | .NET 4 |
| - | .NET 4.5.2 | .NET 3.5 Client Profile |
| - | .NET 4.5.1 | .NET 3 |
| - | .NET 4.5 | - |
